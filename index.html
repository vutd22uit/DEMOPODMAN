What is Podman?

Podman is a daemonless container engine for developing, managing, and running OCI Containers on your Linux System. It was created to ease the transition from Docker and allows running rootless containers.

Installation

Install Podman on your system using the following commands:

sudo apt-get update
sudo apt-get -y install podman

For other distributions, refer to the official Podman installation guide.

Basic Commands

  • Check Podman version: podman --version
  • Pull an image: podman pull IMAGE_NAME
  • Run a container: podman run -d --name CONTAINER_NAME IMAGE_NAME
  • List running containers: podman ps
  • Stop a container: podman stop CONTAINER_NAME

Docker Compatibility

Podman provides a Docker-compatible CLI called podman-docker. You can install it using:

sudo apt-get -y install podman-docker

Advanced Features

Podman includes many advanced features such as rootless containers, pods, and support for Kubernetes YAML. Refer to the official documentation for more details.
